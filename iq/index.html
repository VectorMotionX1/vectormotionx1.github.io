Resource Pack

These are the first resources worth publishing because they solve common beginner-team problems without pretending to be an official rule source.

Checklist

Competition Day Checklist

Pack the boring-but-critical things before leaving for an event.

  • Robot, controller, batteries, charger
  • Tools, spare pins, shafts, wheels, cables
  • Team number plates and inspection notes
  • Notebook link, schedule, food, water
Practice

Skills Practice Log

Turn practice into data instead of vibes.

  • Date, driver, robot version
  • Driver skills and programming skills score
  • Top mistake and next adjustment
  • Best run video link
Scouting

Team Scouting Sheet

Capture useful match observations without overloading students.

  • Team number, event, match
  • Scoring strengths and reliability
  • Driver style and alliance fit
  • One thing to ask in the pits
Notebook

Notebook Entry Prompts

Prompts that help students document their own process.

  • What problem did we notice?
  • What options did we compare?
  • What did we test?
  • What changed because of the result?
Robot

Build Review Checklist

A quick review before practice or inspection.

  • Loose shafts, stressed pins, cable routing
  • Battery access and controller pairing
  • Repeatable scoring path
  • Known weak points and spare parts
Strategy

Match Plan Sheet

Agree on a simple plan before the match starts.

  • Starting position
  • First 20 seconds plan
  • Partner role and backup plan
  • Endgame priority

Beginner Roadmap

A simple path for teams that are early in the season and need structure more than theory.

1

Understand the game

  • Read the scoring summary
  • Watch one match video
  • List the easiest points
2

Build a reliable base

  • Drive straight
  • Turn consistently
  • Survive hard practice
3

Practice measurable runs

  • Use a timer
  • Record scores
  • Fix one weakness per session
4

Prepare for the event

  • Pack and inspect
  • Plan matches
  • Review after every run
